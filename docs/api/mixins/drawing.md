# Drawing Mixin

Creates geometric entities in the active CAD drawing via COM.

Supported entity types: line, circle, arc, rectangle, polyline, text, spline, hatch, ellipse, dimension, leader/mleader.

    options:
      show_source: false
      members_order: source
